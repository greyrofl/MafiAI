# MafiAI | AI-Powered Mafia

**MafiAI** — это современная платформа для игры в «Мафию», где грань между человеком и искусственным интеллектом стирается. Проект создан в рамках хакатона для демонстрации возможностей LLM в ролевых социальных сценариях.

---

## Ключевые особенности

- **Интеграция AI-агентов**: В каждой партии участвуют боты на базе **YandexGPT**, обладающие уникальными характерами и стратегиями поведения.
- **Neural Memory (Сжатие контекста)**: Уникальная механика суммаризации игровых дней. ИИ не просто «читает чат», он помнит ключевые обвинения и события прошлых раундов благодаря динамическому сжатию истории.
- **Динамическая смена фаз**: Плавные визуальные переходы между режимами «День» и «Ночь» с изменением логики доступных действий.
- **🕵️‍♂️ Панель наблюдателя (Admin View)**: Возможность заглянуть "под капот" ИИ — видеть скрытые мысли ботов и их реальные цели в режиме реального времени.
- **💎 Material Design 3**: Современный минималистичный дизайн с использованием мягких градиентов и эффекта матового стекла.

## Технологический стек фронтэнда

- **Frontend**: React 18, Vite (сборка).
- **Styling**: Tailwind CSS (Custom Soft UI palette).
- **Icons**: Lucide React.
- **AI**: YandexGPT API (анализ контекста, генерация реплик, саммаризация).
- **Real-time**: Socket.io для мгновенного взаимодействия.

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## Сборка проекта

```bash
npm run build
```

## Структура приложения

- Главная страница: Вход в систему и выбор режима.
